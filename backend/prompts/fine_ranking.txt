## 你的任务

通读上面的视频描述，感受这段视频的灵魂——它的节奏、情绪、画面感、叙事走向。
然后从候选中选出最契合的，按匹配度从高到低排序。

判断时，把你感受到的视频气质，和每首候选的风格、情绪、场景、文化语境、能量形状、乐器质感等特征做整体性的匹配。
不要机械对照字段，而是像一个导演选曲一样：想象这段视频配上这首BGM，观众会有什么感受？

## 输出要求

必须输出严格的JSON格式，不要有任何其他文字说明：

```json
{
  "rankings": [
    {
      "rank": 1,
      "bgm_id": "bgm_xxxxxxxx",
      "score": 0.85,
      "reason": "用一两句话说明为什么这首最配，从画面感、情绪、节奏等角度",
      "recommended_start_sec": 0,
      "start_sec_reason": "从哪里切入最合适，为什么"
    }
  ],
  "overall_note": "整体选曲思路的简要说明"
}
```

**注意**：
- 排序必须包含所有候选BGM
- score范围0-1，越高越匹配
- reason要具体，说明这首BGM和视频的契合点在哪里
- recommended_start_sec必须在BGM的duration范围内
- 部分BGM字段可能为空，根据可用数据判断
